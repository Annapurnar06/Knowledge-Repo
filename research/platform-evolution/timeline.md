Factory's trajectory — and the broader shift from AI assistants to autonomous factories — can be traced through a series of inflection points. Understanding the timeline clarifies why specific architectural decisions were made and where the industry is heading.

## Factory Origins

Factory was founded in 2023 by **Matan Grinberg** and **Eno Reyes**, who met at a LangChain hackathon. Their backgrounds set the technical DNA of the company:

- **Matan Grinberg**: Princeton undergrad (studied under Juan Maldacena, the AdS/CFT theorist), UC Berkeley PhD dropout. Deep theoretical grounding applied to systems thinking.
- **Eno Reyes**: Cognitive science at Princeton, ML engineer at Hugging Face, software engineer at Microsoft. Production ML experience across the stack.

Factory is Sequoia-backed with a **$50M Series B at $300M valuation** from NEA, Sequoia, J.P. Morgan, and Nvidia — signaling institutional conviction that autonomous software engineering is a venture-scale category.

## Broader Industry Timeline

The AI coding landscape evolved through distinct phases, each enabling the next:

| Date | Event | Significance |
|------|-------|-------------|
| **2021** | GitHub Copilot | Code completion enters the mainstream; proves LLMs can write useful code |
| **Mar 2023** | GPT-4 | Step-function improvement in reasoning; makes agentic coding plausible |
| **Mar 2023** | AutoGPT | First viral autonomous agent experiment; demonstrates the vision (and the failure modes) |
| **Mar 2024** | Devin AI | First "AI software engineer" branding; forces the industry to take autonomous coding seriously |
| **Feb 2025** | Claude Code | Anthropic ships a terminal-native coding agent; validates the CLI-first pattern |
| **Jul 2025** | GPT-5 (256K context) | The "OH SHIT I CAN USE THIS" moment — context windows large enough for real codebases |
| **Jan 2026** | GasTown | Multi-agent coordination tools emerge |
| **Feb 2026** | Attractor dark factory | First fully autonomous "dark factory" — agents operating without human oversight loops |

## Founder Takeaway

The timeline reveals a consistent pattern: each capability unlock (better models, larger context, better tooling) enabled a new class of autonomous system. For vertical AI builders, the lesson is that your architecture must be designed to absorb these step-function improvements — the companies that won each phase were the ones whose systems could immediately exploit the next foundation model upgrade rather than requiring a rewrite.
